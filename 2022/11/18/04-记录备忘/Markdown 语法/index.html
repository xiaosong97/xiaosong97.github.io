Markdown 语法

VSCode 中预览 Markdown ⇧⌘V
Markdown PDF 插件将 md 文件转为 pdf: “>export”

标题

  • # 开头为一级标题

  • ## 开头则为二级标题

  • 依此类推

段落

换行

行尾使用两个以上的空格加上回车，否则可能无法成功换行。

字体

  1. 斜体文字：使用1个 * 或者 _ 来包围某几个字，则这几个字为 斜体

  2. 粗体：使用2个 * 或者 _ 来包围某几个字，则这几个字为 粗体

  3. 粗斜体：使用3个 * 或者 _ 来包围某几个字，则这几个字为 粗斜体

分割线

使用三个以上的星号 * 、减号-、下划线_ 可以建立一个分割线

删除线

文字的两端各加上两个波浪线即可 ~

要删除的文字

下划线

借助 HTML 的标签实现

带下划线的文本

脚注

脚注是对文本的补充说明

Markdown 脚注的格式如下：

列表

有序列表和无序列表

无序列表使用 *、+、- 作为标记

星号：

  • item1

  • item2

加号：

  • item1

  • item2

减号：

  • item1

  • item2

有序列表使用数字来作为标记

  1. item1

  2. item2

列表嵌套

列表嵌套只需在子列表的选项前添加四个空格即可：

  1. 第一项

    • item1.1

    • item1.2

  2. 第二项

    • item2.1

    • item2.2

  3. 第三项

    1. item3.1

Markdown 区块

区块引用是在段落开头使用 > 符号，然后后面紧跟一个空格符号

我是引文

我是第一层引用

我是第二层引用

我是第三层引用

这样也可以

代码块

使用 ``` 来包围多行代码块（代码前后各三个 `）

这是一个多行代码的示例

单行代码使用一个 ` 即可，如 test()

链接

链接使用方法如下：

[链接名称](链接地址)

或者 <链接地址>

例如：

百度

https://www.baidu.com

高级链接

也就是用变量来代替具体的链接地址，然后在文档尾部统一定义这些变量的具体网址

百度的url：baidu

图片

Markdown 图片的语法格式为:

![alt 属性文本](图片地址)

属性文本为图片的代替文字

表格

Markdown 表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行

表头1 表头2
单元格11 单元格12
单元格21 单元格22

对齐方式：

-: 设置本列右对齐

:- 设置本列左对齐

:-: 设置本列居中对齐
